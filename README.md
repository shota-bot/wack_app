# 利用方法
ユーザー登録することで、LIVEやイベントで撮った写真をアップロードすることができます。
ユーザー登録しなくても、画像一覧ページの閲覧、画像の検索は可能です。
ユーザー登録したユーザーは、写真の投稿、投稿内容の編集、削除、他ユーザーのフォロー、グッズ交換相手の募集をすることができます。グッズを出品する際には、自分が求めるグッズもしくは金額を設定し、手渡しなのか郵送可なのかなどを提示します。欲しいグッズを見つけたユーザーは、出品したユーザーにリクエストを送ることができ、出品者と個人チャットが可能になります。交換日時や受け渡し場所など、詳細な相談をすることができます。

# 目指した課題解決
撮った写真を共有したい。
推しのメンバーの画像をまとめてみたい。
グッズを交換してもらえる人を募集したい。
